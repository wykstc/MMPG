# KORP pairwise potential

Compute knowledge-based pairwise potential for pdb

Download score file `korp6Dv1.bin` from：
https://chaconlab.org/modeling/korp/down-korp/item/korp-linux

## Usage

```
./process_korp.sh <INPUT_ROOT> <OUTPUT_ROOT>"
```
